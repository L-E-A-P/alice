Questo è un esempio di draft.

## Come funziona

I file nella cartella `_drafts/` non vengono pubblicati automaticamente:
- Non serve la data nel nome del file
- Per vederli in locale: `bundle exec jekyll serve --drafts`
- Quando sei pronto a pubblicare, sposta il file in `_posts/` e rinominalo con la data

## Workflow

1. Scrivi il tuo contenuto qui in `_drafts/`
2. Testa in locale con `--drafts`
3. Quando è pronto, sposta in `_posts/` come `YYYY-MM-DD-titolo.md`
